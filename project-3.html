NuCamp

Camping website used discover and to reserve campsites.

Live Link

Project Overview

I used JavaScript in the form jQuery to display a modal dialog box when the user clicks on the "Login" button. The dialog box contains a form for the user to enter their username and password. When the user clicks on the "Reservation" button, a modal dialog box is displayed containing a form for the user to enter their desired date, number of campers, and campsite type.

Happy camping!
